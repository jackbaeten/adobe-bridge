# Adobe Bridge

Connects InDesign, Illustrator and Photoshop directly to After Effects.
Works like Overlord — background tray app with CEP panels in each Adobe app.

## Download

Go to **Actions** tab → click the latest build → download **Adobe-Bridge-Setup**.

## What it does

- Runs in the system tray (starts with Windows)
- Push selected items from InDesign to After Effects with one click
- New comp created automatically with correct dimensions
- All layers transferred with correct position, rotation, opacity
- Bridge Manager lets you install/uninstall connections per app
